# Calculator

## Overview

A simple command-line calculator built using Python. This project performs basic arithmetic operations based on the user's choice.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* User-friendly menu

## Technologies Used

* Python

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Select an operation from the menu.
4. Enter two numbers to get the result.
