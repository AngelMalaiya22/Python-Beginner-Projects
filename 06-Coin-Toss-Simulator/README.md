# Coin Toss Simulator

## Overview

A simple Python program that simulates tossing a coin. Each toss randomly returns either **Heads** or **Tails**, and users can continue tossing as many times as they like.

## Features

* Randomly generates Heads or Tails
* Interactive user interface
* Supports multiple coin tosses
* Simple and beginner-friendly

## Technologies Used

* Python
* Random module

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Press **Enter** to toss the coin.
4. Choose whether to toss again.
