# Bank Account System

## Overview

A simple Python Bank Account System that demonstrates the use of Object-Oriented Programming (OOP). Users can create a bank account, check their balance, deposit money, and withdraw money through an interactive menu.

## Features

* Create a bank account
* Check account balance
* Deposit money
* Withdraw money
* Uses Object-Oriented Programming (Classes and Objects)
* Menu-driven interface

## Technologies Used

* Python

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Enter the account holder's name.
4. Choose an option from the menu to manage the account.
