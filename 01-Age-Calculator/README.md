# Age Calculator

## Overview

A simple Python program that calculates the user's age based on their date of birth. The program uses the current date to determine the accurate age in years.

## Features

* Calculates age from date of birth
* Uses the current system date
* Accurate age calculation
* Simple and interactive interface

## Technologies Used

* Python
* Datetime module

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Enter your date of birth in **YYYY-MM-DD** format.
4. View your calculated age.
