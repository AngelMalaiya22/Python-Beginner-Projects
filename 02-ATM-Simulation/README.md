# ATM Simulation

## Overview

A simple Python program that simulates basic ATM operations. Users can check their account balance, deposit money, withdraw money, and exit the application through a menu-driven interface.

## Features

* Check account balance
* Deposit money
* Withdraw money
* Prevents overdrawing the account
* Interactive menu-driven interface

## Technologies Used

* Python

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Select an option from the ATM menu.
4. Perform banking operations until you choose to exit.
