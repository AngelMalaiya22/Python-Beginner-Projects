# BMI Calculator

## Overview

A simple Python program that calculates the Body Mass Index (BMI) based on the user's weight and height. It also classifies the BMI into standard health categories.

## Features

* Calculates Body Mass Index (BMI)
* Displays BMI value with two decimal places
* Classifies BMI as Underweight, Normal weight, Overweight, or Obese
* Simple and easy-to-use interface

## Technologies Used

* Python

## How to Run

1. Open the Jupyter Notebook.
2. Run the code cell.
3. Enter your weight in kilograms and height in meters.
4. View your BMI and health category.
